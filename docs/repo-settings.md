# Repository Settings

Use these commands after authenticating `gh` with permission to edit the repository.

## Description

```bash
gh repo edit Frank-Li-Yixuan/gh-flakefinder --description "Find GitHub Actions jobs that fail, get rerun, and mysteriously pass."
```

## Topics

```bash
gh repo edit Frank-Li-Yixuan/gh-flakefinder \
  --add-topic github-actions \
  --add-topic gh-extension \
  --add-topic cli \
  --add-topic ci \
  --add-topic devops \
  --add-topic flaky-tests \
  --add-topic testing-tools \
  --add-topic golang \
  --add-topic open-source \
  --add-topic developer-tools
```

## Social Preview

GitHub repository settings can use `assets/social-preview.svg` as a source design for a social preview image. If GitHub requires PNG upload, export the SVG to PNG locally before uploading through the repository settings UI.

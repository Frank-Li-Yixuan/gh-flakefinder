# Social Preview Upload

## Source File

Use `assets/social-preview.svg` as the source design for the repository social preview.

No PNG was generated during the 48-hour checklist because no local SVG conversion tool was available without adding project dependencies. If exporting manually, use:

- format: PNG
- target dimensions: approximately 1280x640
- file size: under 1 MB

## Why It Matters

The social preview is what people see when the repository URL is shared in chats, social posts, and link previews. A clear preview helps first-time visitors recognize that this is a GitHub CLI extension for finding likely flaky GitHub Actions runs.

## GitHub UI Path

1. Open https://github.com/Frank-Li-Yixuan/gh-flakefinder.
2. Go to Settings.
3. Open the Social preview section.
4. Select Edit.
5. Upload the PNG exported from `assets/social-preview.svg`.
6. Save the settings change.
